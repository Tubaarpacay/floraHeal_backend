# FloraHeal Backend

Backend API for a plant tracking application developed as a university group project.

## Description

This project allows users to:
- Add and manage plants
- Track water and fertilizer entries
- Store and retrieve user data
- Add, view and delete records

## Technologies Used

- Node.js
- Express.js
- MongoDB (Mongoose)
- JavaScript

## Installation

1. Install dependencies:
   npm install

2. Create a `.env` file in the root directory:
   MONGO_URI=your_mongodb_connection_string
   PORT=5000

3. Run the server:
   npm start
